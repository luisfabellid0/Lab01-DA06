Páginas web responsivas agradables

Hecho con HTML y CSS

Responsividad

Una página web responsiva es

aquella que se adapta a

cualquier tamaño de pantalla.

Influye en la experiencia del usuario

y en la marca de la página.

CSS

Lenguaje que maneja el diseño

y presentación de las páginas web.

Permite separar el contenido

HTML de su presentación visual.

Utilizable en todos los navegadores.

Diseño

El diseño web se enfoca en la apariencia

y la estructura de una página. Incluye

la selección de colores, tipografías,

y la disposición de los elementos.

Un buen diseño mejora la estética

y facilita la navegación para los usuarios.
